Access an element of a pod_tuple by type

The same as std::get on an std::tuple.

Note that these functions are available, both, in the bio namespace and in namespace std. As is the case with std::tuple, this function is only defined if the type appears once in the tuple, i.e. std::get<int>(std::tuple<int, int>{1,2}) is not defined.
