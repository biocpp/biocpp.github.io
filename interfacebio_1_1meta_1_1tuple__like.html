Detailed Description

Whether a type behaves like a tuple.

Types that meet this concept are for example std::tuple, std::pair, std::array, bio::meta::pod_tuple, bio::meta::record. The std::totally_ordered will only be required if all types contained in the tuple-like data structure are themselves strict totally ordered.
