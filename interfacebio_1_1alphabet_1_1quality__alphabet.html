Detailed Description

A concept that indicates whether an alphabet represents quality scores.

In addition to the requirements for bio::alphabet::alphabet, the quality_alphabet introduces a requirement for conversion functions from and to a Phred score.

Requirements

  1. t shall model bio::alphabet::alphabet
  2. bio::alphabet::to_phred needs to be defined for objects of type t

See the documentation pages for the respective requirements.
