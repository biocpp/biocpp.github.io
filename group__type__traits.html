Returns the number of times you can call bio::value_type_t recursively on t (type trait).

Template Parameters
tThe type to be queried; must resolve bio::value_type_t at least once.

Attention, this type trait implicitly removes cv-qualifiers and reference from the types it recurses on and returns.
